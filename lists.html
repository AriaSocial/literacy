HTMLリストの種類

順序なしリスト (ul)

買い物リストなど、順序が関係ない項目に使用。

  • りんご
  • バナナ
  • オレンジ
  • 牛乳

順序付きリスト (ol)

手順やランキングなど、順序が重要な項目に使用。

  1. まず、手を洗います。
  2. 次に、材料を準備します。
  3. そして、調理を開始します。
  4. 最後に、美味しくいただきます。

説明リスト (dl)

用語とその説明を定義するのに使用。

HTML
HyperText Markup Language（ハイパーテキスト マークアップ ランゲージ）の略。ウェブページを作成するための基本的な言語です。
CSS
Cascading Style Sheets（カスケーディング スタイル シート）の略。ウェブページの見た目やレイアウトを定義するための言語です。
JavaScript
ウェブページに動的な機能を追加するためのプログラミング言語です。
